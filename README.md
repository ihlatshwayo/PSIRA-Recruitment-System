# PSIRA Recruitment System

## Setup Instructions

### Backend Setup:
1. Navigate to Backend folder
2. Run: `dotnet restore`
3. Update connection string in appsettings.json
4. Run: `dotnet run`

### Frontend Setup:
1. Navigate to Frontend folder
2. Run: `npm install`
3. Run: `ng serve`

### Database Setup:
1. Execute schema.sql in MySQL
2. Update connection string in backend

## Default Admin Login:
- Username: admin
- Password: HRPassword

## Features:
- Admin job post management
- Applicant registration and job applications
- CV sifting with automatic scoring
- File upload for CVs
